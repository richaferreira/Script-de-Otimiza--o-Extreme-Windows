# 🎨 Extreme Optimizer v3.5 (Stylist HUB)

Esta versão traz o poder da personalização estética para o Windows 11, integrando conceitos do **Ramen Software Styling Guide** e do motor **Windhawk**.

## 🚀 O que há de novo na v3.5 (Stylist HUB)?

| Recurso | Descrição |
| :--- | :--- |
| **🎨 Windhawk Engine** | Opção direta para instalar o Windhawk, o motor essencial para aplicar temas como Mica, Arredondamento e Transparência total. |
| **📏 Taskbar Scaling** | Ajuste do tamanho da barra de tarefas (Pequena/Média/Grande) via registro. |
| **📐 Alinhamento Flexível** | Alternância rápida entre o alinhamento central (Win11) e à esquerda (Win10). |
| **✨ Transparência OLED** | Habilitação de flags de registro para aumentar a transparência nativa. |
| **🛡️ Gamer & Dev Core** | Mantém todas as otimizações de latência (TCPNoDelay) e produtividade da v3.0. |

---

## 🛠️ Como Personalizar sua Barra de Tarefas

1.  **Estilo Nativo**:
    *   Use as opções **[S]**, **[A]**, **[C]** e **[T]** para ajustes rápidos de tamanho, alinhamento e transparência sem precisar de softwares externos.

2.  **Estilo Avançado (Windhawk)**:
    *   Selecione a opção **[W]** para instalar o Windhawk.
    *   Dentro do Windhawk, procure pelo mod **"Windows 11 Taskbar Styler"**.
    *   Você poderá importar temas como o **"SimplyTransparent"** ou **"DockLike"** diretamente das referências do Ramen Software.

---

## 💡 Dicas de Estilo

*   **Barra de Tarefas Pequena**: Ideal para quem usa monitores menores ou quer mais espaço na tela.
*   **Transparência**: O Windows 11 tem limites nativos; para uma transparência "vidro" total, o uso do Windhawk com o mod de Taskbar Styler é altamente recomendado.
*   **Reinicialização**: O script reinicia o `explorer.exe` automaticamente para aplicar as mudanças de UI.
