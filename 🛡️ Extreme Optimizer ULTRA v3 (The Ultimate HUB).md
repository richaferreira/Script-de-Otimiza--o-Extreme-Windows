# 🛡️ Extreme Optimizer ULTRA v3 (The Ultimate HUB)

Esta é a versão definitiva do script de otimização, integrando segurança avançada, ferramentas lendárias do **Chris Titus Tech** e ajustes finos para **Gamers e Desenvolvedores**.

## 🚀 O que há de novo na v3.0 (ULTRA HUB)?

| Recurso | Descrição |
| :--- | :--- |
| **📁 Registro de Logs** | Todas as ações são salvas em `%ProgramData%\ExtremeOptimizer\logs` para auditoria. |
| **🛡️ Backup de Registro** | Exportação automática de chaves críticas (`.reg`) antes de modificações. |
| **⚡ TCPNoDelay** | Ajuste fino de rede para reduzir latência em jogos online. |
| **📁 LongPaths** | Suporte a caminhos de arquivos longos (essencial para desenvolvedores Node/Git). |
| **🔋 High Performance** | Ativação automática do plano de energia de alto desempenho via `powercfg`. |
| **🔄 Restore Total** | Opção dedicada para reverter telemetria, serviços e planos de energia. |

---

## 🛠️ Como Utilizar o HUB

1.  **Execução**:
    *   Clique com o botão direito no script `extreme_optimizer_v3_ultra.ps1` e selecione **Executar com o PowerShell**.
    *   O script solicitará privilégios de **Administrador** automaticamente.

2.  **Fluxo Recomendado**:
    *   **Opção [1] (COMBO ULTRA)**: O "Botão do Pânico" para quem quer tudo pronto. Faz backup, cria ponto de restauração e aplica todas as otimizações (Gamer, Dev e Debloat).
    *   **Opção [2] (GAMER)**: Focado apenas em latência e energia.
    *   **Opção [3] (DEVELOPER)**: Ativa WSL2, Modo Dev e LongPaths.
    *   **Opção [R] (RESTORE)**: Use esta opção se algo não funcionar como esperado; ela importará seus backups de registro.

---

## ⚠️ Avisos e Dicas

*   **Ponto de Restauração**: A opção [1] cria um ponto de restauração automaticamente. Não pule esta etapa!
*   **WSL2**: A ativação do WSL2 requer que a **Virtualização** esteja habilitada na sua BIOS.
*   **Debloat**: Este script usa uma lista de remoção "segura", mantendo a Loja e o Xbox funcionando perfeitamente.
