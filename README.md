# 🚀 Script de Otimização Extreme Windows 11 (v2.0) - Gamer & Developer

Este script foi atualizado para integrar as ferramentas lendárias do **Chris Titus Tech (CTT)**, combinando ajustes manuais de baixa latência com a facilidade da interface gráfica oficial.

## 🛠️ O que há de novo na v2.0?

| Recurso | Descrição |
| :--- | :--- |
| **🎮 Chris Titus Utility** | Acesso direto ao comando `irm christitus.com/win | iex` via menu. |
| **🛠️ MicroWin** | Atalho para criar uma ISO ultra-enxuta (MicroWin) do Windows. |
| **⚡ Ultimate Combo** | Executa os ajustes nativos de latência e abre o utilitário do CTT para o debloat final. |
| **🛡️ VBS/HVCI Toggle** | Desativação simplificada da "Integridade de Memória" para boost de FPS. |

---

## 📋 Como Usar

1.  **Execução**:
    *   Abra o PowerShell como **Administrador**.
    *   Rode o script: `.\extreme_optimizer_v2.ps1`

2.  **Opções Recomendadas**:
    *   **Opção [1]**: Sempre crie um ponto de restauração primeiro.
    *   **Opção [T]**: Abre a interface gráfica do Chris Titus Tech para instalar programas (Winget) e fazer tweaks visuais.
    *   **Opção [4]**: Aplica os ajustes de rede e prioridade de CPU para Gamers.
    *   **Opção [8]**: O caminho completo para quem quer o sistema mais rápido possível.

---

## 💡 Dicas de Performance

*   **Rede**: O script desativa o *Nagle's Algorithm*, o que reduz o ping em jogos online, mas pode impactar levemente a velocidade de download de arquivos muito grandes.
*   **VBS (Segurança vs Performance)**: Desativar o VBS (Opção 6) pode dar um ganho de até 10% em FPS, especialmente em processadores Intel de 10ª geração ou inferiores e AMD Ryzen série 3000 ou inferior.
*   **Debloat**: Se você usa o **Xbox Game Pass**, use a interface gráfica do Chris Titus (Opção T) e marque a caixa "Keep Xbox" para não quebrar a loja.
